# madness_2026
ML-powered March Madness 2026 bracket predictor &amp; head-2-head matchups. LightGBM + Logistic Regression ensemble trained on 15 years of Barttorvik T-Rank efficiency data (944 tournament games). Streamlit web app with Monte Carlo simulation, adjustable confidence gears, projected scores, and full statistical comparison for all 68 tournament teams.
